Tentang Hutao.id

Hutao.id adalah proyek web sederhana bertema anime & manga yang dibuat menggunakan Jikan API. Website ini memungkinkan pengguna untuk mencari anime atau manga favorit, melihat detail lengkap, trailer, rekomendasi, dan filter genre â€” semuanya dalam antarmuka yang bersih dan responsif (at least it's works).

Tujuan Proyek

Website ini dibuat sebagai bagian dari tugas akhir pelatihan HTML. Fokus utama pengembangan adalah struktur kode yang modular, pemisahan logika dan styling, serta penggunaan API eksternal secara efisien.

Fitur Utama

  • Pencarian Anime & Manga
  • Halaman detail per judul
  • Trailer anime (jika tersedia)
  • Filter berdasarkan genre & tipe
  • Navigasi halaman (pagination)
  • Simulasi fitur nonton (tidak streaming)

Teknologi yang Digunakan

  • HTML5, CSS3, JavaScript
  • Jikan API v4 (open source)
  • Desain responsif (desktop-friendly)
